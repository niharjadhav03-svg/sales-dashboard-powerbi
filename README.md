# 📊 Sales Performance Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Sales Performance Dashboard** developed using Power BI.
It analyzes transactional sales data to uncover trends, monitor key performance indicators (KPIs), and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales and profitability
* Identify top-performing regions and product categories
* Track monthly sales trends and patterns
* Highlight areas of low profitability

---

## 📁 Data Sources

* **Orders.csv** – Contains transactional sales data (sales, quantity, region, etc.)
* **Details.csv** – Includes product and category-level information

---

## 📊 Key Features

* 📈 Interactive dashboard with dynamic filters
* 🌍 Region-wise sales and profit analysis
* 📦 Category and product performance insights
* 📅 Time-based trend analysis (monthly/periodic)
* 💡 Clear visualization of KPIs (Sales, Profit, Quantity)

---

## 🛠 Tools & Technologies

* **Power BI Desktop** – Data visualization and dashboard creation
* **Microsoft Excel / CSV** – Data storage and preprocessing

---

## 📈 Key Insights

* The **West region** contributes the highest share of total sales
* Certain categories show **high revenue but lower profit margins**
* Sales trends indicate **seasonal fluctuations** across months

---

## 🚀 How to Run the Project

1. Download the `.pbix` file from this repository
2. Open it using Power BI Desktop
3. Ensure the dataset files (CSV) are correctly linked
4. Interact with the dashboard to explore insights
